# visual-config-diff Specification

## Purpose
可视化配置对比页面：按实例分组展示映射结果，两层结构（实例总览 → 监听器字段对比），JSON 对比放侧边抽屉，不兼容项自动推荐替代方案。

## ADDED Requirements

### Requirement: 实例分组总览
系统 SHALL 将映射结果按源端→目标端实例分组展示，每组显示统计摘要。

#### Scenario: 展示实例分组
- **WHEN** 映射完成后进入配置对比步骤
- **THEN** 系统按"源端实例→目标端实例"分组展示，每组标题显示实例名称和统计（N项映射 ✓、M项需确认 ⚠、K项不兼容 ✗）

#### Scenario: 展开实例分组
- **WHEN** 用户点击某个实例分组
- **THEN** 展开该组下所有监听器和转发规则的字段级对比卡片

### Requirement: 监听器字段级对比
系统 SHALL 为每个监听器展示字段级的源端→目标端对比，每行一个配置字段。

#### Scenario: 完全映射的字段
- **WHEN** 配置字段可直接映射
- **THEN** 展示 "源值 → 目标值 ✓"，绿色标记

#### Scenario: 需确认的字段（有推荐替代）
- **WHEN** 配置字段映射有变化且系统有推荐替代方案
- **THEN** 展示 "源值 → [推荐值 ▼] ⚠"，橙色标记，目标值为可修改的下拉框

#### Scenario: 不兼容的字段
- **WHEN** 配置字段无法映射且无替代方案
- **THEN** 展示 "源值 → 不兼容 ✗"，红色标记，说明原因

### Requirement: 转发规则平铺展示
系统 SHALL 将转发规则平铺在所属监听器下方展示。

#### Scenario: 展示转发规则
- **WHEN** HTTP/HTTPS 监听器有转发规则
- **THEN** 在该监听器对比卡片下方直接展示所有转发规则的对比行

### Requirement: JSON 侧边抽屉
系统 SHALL 提供 JSON 对比的侧边抽屉，展示源端和目标端的原始/映射后 JSON。

#### Scenario: 打开 JSON 抽屉
- **WHEN** 用户点击实例分组标题的 [JSON] 按钮
- **THEN** 右侧滑出抽屉，上下分栏展示阿里云原始配置 JSON 和腾讯云映射后配置 JSON

#### Scenario: 关闭 JSON 抽屉
- **WHEN** 用户点击抽屉关闭按钮或遮罩区域
- **THEN** 抽屉滑回隐藏

### Requirement: 统计卡片
系统 SHALL 在页面顶部展示映射统计卡片。

#### Scenario: 展示统计
- **WHEN** 映射结果加载完成
- **THEN** 展示四个统计卡片：总项、完全映射、需确认、不兼容
